# Moora Website

## Proyecto

Aplicación web para seguimiento de objetivos y journaling diario con IA.

## Cómo ejecutar el proyecto localmente

Requisitos: Node.js & npm instalados - [instalar con nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Pasos:

```sh
# Paso 1: Navega al directorio del proyecto
cd moora_website

# Paso 2: Instala las dependencias
npm install

# Paso 3: Inicia el servidor de desarrollo
npm run dev
```

La aplicación estará disponible en http://localhost:8080

## Tecnologías utilizadas

Este proyecto está construido con:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
